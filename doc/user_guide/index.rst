User guide
==========

TensorLy-Torch was written to provide out-of-the-box Tensor layers that can be readily used within any PyTorch network or code.
It builds on top of `TensorLy <tensorly.org>`_ and enables anyone to use tensor methods within deep networks, even without a deep knowledge of tensor algebra.
