tltorch._tensor_dropout.CPDropout

class tltorch._tensor_dropout.CPDropout(proba, min_dim=1)[source]

Decomposition Hook for Tensor Dropout on Tucker tensors

Parameters:
pfloat, probability of dropout
min_dimint

Minimum dimension size for which to apply dropout. For instance, if a tensor if of shape (32, 32, 3, 3) and min_dim = 4 then dropout will not be applied to the last two modes.
